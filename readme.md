### 特性：

1. 支持输入API KEY（默认会读取插件目录下的api_key.txt中的内容作为API密钥）
2. 支持自定义API地址
3. 支持设定分辨率和宽高比
4. 支持设置代理地址
5. 支持启用谷歌搜索
6. 更新随机种子，欺骗comfyui可以在不改变参数的情况下多次运行

### 使用方法：

- 在Custom nodes文件夹中新建一个英文文件夹，把__init__.py文件复制进去就行
- https://grsai.ai/zh 这个网站可以买API
